# -Cab-Trips-Analytics-on-Databricks-Lakehouse-DLT-LakeFlow-Medallion-Architecture-
An end-to-end Databricks Lakehouse project that processes cab trip data (~500K+ records) using Declarative Pipelines (Delta Live Tables / LakeFlow) with Full + Incremental Loads, Auto Loader streaming ingestion, and Bronze → Silver → Gold layered architecture.
